# watanabe-reserch-server

## Features

LEDCubeで光らせたい部分をスプレッドシートに一時的に保存し、返すAPI

## Requirement

* Node.js v17.x
* Yarn 1.22.10
* clasp 2.4.1
* TypeScript 3.8.2

## Installation

### Node.js

以下のリンクからNode.js v17.xをダウンロードしてください。
<https://nodejs.org/ja/download/>

### Yarn

```bash
# npm 経由でyarnをインストール
npm install -g yarn
# yarnのバージョンを確認
yarn -v
```

## Usage

```bash
clasp login
clasp push
```
